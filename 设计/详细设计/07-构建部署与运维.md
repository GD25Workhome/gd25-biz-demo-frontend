# 构建部署与运维
环境说明：本文以 macOS 开发环境为准。

## 开发与构建命令
- 开发：`npm run dev`（Vite 开发服务器）
- 构建：`npm run build`（产物输出至 `dist/`）
- 预览：`npm run preview`

## 环境注入
- 使用 `.env`/`.env.local` 管理 `VITE_API_BASE_URL` 等变量
- 生产部署前确保 `baseURL` 指向正式后端地址

## 部署方式
- 静态托管：Nginx/Apache/静态主机（Netlify、Vercel）
- 路由策略：Browser Router 需服务端配置 fallback 到 `index.html`

## CORS 与代理
- 正式环境优先由后端开启 CORS
- 开发环境使用 Vite 代理，避免跨域影响联调

## 性能建议
- 按需加载：路由级代码分割（`lazy`）
- 资源优化：启用 gzip/br 传输、缓存策略
- 监控：前端埋点与错误上报（可后续引入）

## 问题排查
- 网络：检查 `baseURL`、代理与响应码
- 构建：确认环境变量在构建阶段正确注入
- 运行：浏览器控制台错误与 Network 面板日志
